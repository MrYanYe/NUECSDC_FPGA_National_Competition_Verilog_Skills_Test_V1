加入2025年能力测试题（刚完）

比赛时会在一个教室里，地上有插座，怕不够可以带插线板（插座数量就刚刚够用，因为一屋子坐满了）。赛前说要断网，但其实比赛的教室里和厕所**没有屏蔽信号**的东西，就靠教室里的两个监考老师（看着其实就是大学生志愿者，大概研一的岁数）走着看你作没作弊，但因为教室里大家都在讨论，所以略微嘈杂，并不像真正的考场那样。所以电脑联网没人管，迅速地复制粘贴电脑端微信的文字是可以的，偷偷地在桌子下面用手机也可以，但监考的会着重看你用手机。

考试前2分钟会把题发到队长QQ群里，然后你做完了就可以让两个监考员来验收，它不会告诉你哪错了，如果你波形对了就会在他的表格上给你的队伍打个勾，然后就可以走了。似乎不完全对也可以，可能要扣点分？我不清楚。

因为考着考着就有队伍做完了，然后陆续往外走，这时候考场纪律是最松的。如果实在做不出来，可以淡定地假装去上厕所或是装作做完了，往外走，去厕所（怕有监控），AI，微信传输助手或其他方式。回来。注意一定要删除注释，有的监考员会看你代码，可能让你简单解释（那种闲的严格的监考员）。不删注释一眼就看出来是AI写的。如果观察两个监考员有一个好说话一个不好说话，可以等好说话（或者只验收波形就完事了的）的那个监考员走近你的时候让他验收。
