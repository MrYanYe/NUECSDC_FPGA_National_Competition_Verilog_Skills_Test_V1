文章介绍了如何用Verilog语言设计有符号和无符号数的加减法器，并组合成一个通用计算器，处理加法、减法、溢出和进位/借位情况。

相关阅读

数字IC基础https://blog.csdn.net/weixin_45791458/category_12365795.html?spm=1001.2014.3001.5482

        本文是对数字IC基础：有符号数和无符号数的加减运算一文中的谈到的有符号数加减法的算法进行Verilog实现，有关算法细节请阅读原文，本文不会过多谈到原理相关问题。

        虽然有符号加减和无符号加减在底层都是使用同样的补码加法器结构，但我们首先分别设计有符号加减法器和无符号加减法器，然后再将其组成一个完整的加减计算单元。

        一个有符号数加减法器的Verilog描述如例1所示。

  1. // 例1
  2. module signed_adder(input signed [7:0] A, B, input mode, output reg signed [7:0] Sum, output reg OF);
  3.     always@(*) begin
  4.         if (mode == 0) begin //有符号加法
  5.             Sum = A + B;
  6.             OF = ((A[7] == B[7]) & (Sum[7] != A[7])); //溢出
  7.         end
  8.         else begin //有符号减法
  9.             Sum = A + (~B) + 1'b1;
  10.             OF = ((A[7] != B[7]) & (Sum[7] != A[7])); //溢出
  11.         end
  12.     end
  13. endmodule

        一个无符号加减法器的Verilog描述如例3所示。

  1. // 例3
  2. module unsigned_adder(input [7:0] A, B, input mode, output reg [7:0] Sum, output reg CF);
  3.     reg cout;
  4.     always@(*) begin
  5.         if (mode == 0) begin //无符号加法
  6.             {cout, Sum} = A + B;
  7.             CF = cout;      //进位
  8.         end
  9.         else begin          //无符号减法
  10.             {cout, Sum} = A + (~B) + 1'b1;
  11.             CF = cout;     //借位
  12.         end
  13.     end
  14. endmodule

        其中A和B会按照算法中谈到的一样，先补零拓展成8位数（这是自动进行的，进一步的位宽拓展问题可以阅读Verilog基础：表达式位宽的确定（位宽拓展）_verilog定义位宽-CSDN博客这篇文章），然后再按照有符号加减的运算。

图2 无符号数加减法器测试的波形

        可以将两者结合，这样一个可以进行有符号数加减法和无符号数加减法的通用计算器就诞生了，如例5所示。

  1. // 例5
  2. module adder(input [7:0] A, B, input [1:0] mode, output reg [7:0] Sum, output reg CF_OF);
  3.     reg cout;
  4.     always@(*) begin
  5.         if (mode == 2'b00) begin //有符号加法
  6.             Sum = A + B;
  7.             CF_OF = ((A[7] == B[7]) & (Sum[7] != A[7])); //溢出
  8.         end
  9.         else if (mode == 2'b01) begin //有符号减法
  10.             Sum = A + (~B) + 1'b1;
  11.             CF_OF = ((A[7] != B[7]) & (Sum[7] != A[7])); //溢出
  12.         end
  13.         else if (mode == 2'b10) begin //无符号加法
  14.             {cout, Sum} = A + B;
  15.             CF_OF = cout;      //进位
  16.         end
  17.         else if (mode == 2'b11) begin //无符号减法
  18.             {cout, Sum} = A + (~B) + 1'b1;
  19.             CF_OF = cout;     //借位
  20.         end
  21.     end
  22. endmodule
